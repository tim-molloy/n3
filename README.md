# N3

## Requirements

* Python3
* pip3

## Installation

`pip3 install -r requirements.txt`

## Usage

### gen.py
```bash 
./gen.py --help
```
Example usage
```bash 
./gen.py --rows 20 --output-path ./data/ --column-data "('int_data', 'integer'), ('string_data', 'string')"
```


### api.py
```bash 
./api.py
```

## Tests

pytest
